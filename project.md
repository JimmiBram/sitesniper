A console based website image screenshooter and scraper that uses the rich console libraries for displaying images. 

Interface: The cli interface has an input field in the bottom for entering a url, a status field (under the input field) with current status. Right of the input field is a dropdown where you can choose scraping debth from 1 to 3. And right of that a number input with max pages. 

Filling up most space is the image area, where screenshots of the website is shown as they come in. 

Output: Each scrape will be stored in /Data/Scrapes as screenshots

User enters a url, it is validated and the scraper starts scraping the frontpage.
