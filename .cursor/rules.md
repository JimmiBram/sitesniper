# Project Rules

## Project
SiteSniper, console based image grapper and crawler

## Tech Stack
- Python 3.14

## Coding Guidelines
- Use type hints
- Follow PEP8
- Prefer small modular functions over large
- Make sure files have clear purpose, refactor if too large or unclear purpose

## Testing
- Use pytest for testing
- Write simple, onpoint tests where we have complexity, not mindless unit testing
- Put tests in tests/ and name them test_*.py
- Add a poetry script for running the tests, poetry run test
